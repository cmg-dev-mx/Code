# Code

## Descripción

Aplicación movil de ejemplo desarrollada con distintas tecnologías nativas y multiplataforma.

## Configuración

### Android

Para ejecutar el proyecto Android, necesitas configurar Firebase:

1. Copia el archivo de ejemplo:
   ```bash
   cp Android/Code/app/google-services.json.example Android/Code/app/google-services.json
   ```

2. Ve a la [consola de Firebase](https://console.firebase.google.com/) y descarga tu archivo `google-services.json` real.

3. Reemplaza el archivo copiado con el archivo real descargado de Firebase.

4. El archivo `google-services.json` está en el `.gitignore` y no se subirá al repositorio por seguridad.
